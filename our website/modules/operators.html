Operators, Operands, and Arithmetic in C

Operators in C are special symbols used to perform operations on variables and values (called operands). The most basic are arithmetic operators, which perform common mathematical operations.

1. Addition (+)

The + operator adds two values together.

int a = 5, b = 3;
int sum = a + b;
printf("Sum: %d\n", sum); // Output: Sum: 8

2. Subtraction (-)

The - operator subtracts the second value from the first. 

int a = 10, b = 4;
int diff = a - b;
printf("Difference: %d\n", diff); // Output: Difference: 6

3. Multiplication (*)

The * operator multiplies two values.

int x = 6, y = 7;
int product = x * y;
printf("Product: %d\n", product); // Output: Product: 42

4. Division (/)

The / operator divides the first value by the second. Note that integer division discards the remainder. 

int a = 20, b = 4;
int result = a / b;
printf("Quotient: %d\n", result); // Output: Quotient: 5

5. Modulus (%)

The % operator gives the remainder of a division operation. 

int a = 10, b = 3;
int mod = a % b;
printf("Remainder: %d\n", mod); // Output: Remainder: 1
Remember operator precedence (like multiplication before addition) to avoid logic errors. Use parentheses for clarity.
