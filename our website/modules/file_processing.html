File Processing

Read from and write to files for data storage and retrieval.

Example

#include 

int main() {
    FILE *f = fopen("data.txt", "w");
    if (f != NULL) {
        fprintf(f, "Hello, file!\n");
        fclose(f);
    }
    return 0;
}
Always check if the file pointer is NULL before using it.
